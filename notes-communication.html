2. Communication & Internet

Networks

A network connects multiple computers to share data.

  • LAN – Local Area Network
  • WAN – Wide Area Network
  • Star, Bus, Ring topologies

Protocols

Rules for data transmission.

  • HTTP – Web communication
  • FTP – File transfer
  • TCP/IP – Internet communication

Network Security

Protects data during transmission.

  • Encryption
  • Firewalls
  • Authentication
